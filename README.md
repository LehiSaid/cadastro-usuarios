# Cadastro de Usuários

Projeto Full Stack para cadastro de usuários.

## Tecnologias

- React
- Vite
- Node.js
- Express
- Prisma ORM
- MongoDB

## Funcionalidades

- Cadastro de usuários
- Listagem de usuários
- Exclusão de usuários

## Como executar

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
